# *public 필드의 문제점
1. public 필드는 *외부 객체에 의해 오염*될 가능성을 열어두는 것
2. 자바 같은 언어에서는 *Get/Set 메소드를 이용하여 내부 필드에 접근하도록 함
3. c#에서는 Get/Set 메소드 뿐 아니라 *필드만큼 편리한 프로퍼티*를 제공
